# Nextra Documentation Template

This is a Nextra-based documentation template.

## Getting Started

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```

## Customization

-   **Theme**: Edit `theme.config.tsx` to change the logo, links, and footer.
-   **Content**: Add your MDX files in the `pages` directory.
-   **Configuration**: Edit `next.config.js` for Next.js and Nextra settings.
