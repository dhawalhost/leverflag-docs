# LeverFlag Documentation

Welcome to the **LeverFlag** documentation. LeverFlag is a powerful, open-source feature flag management service designed for modern development workflows.

## Why LeverFlag?

- **Simple & Fast**: Built with Go for high performance and minimal latency.
- **Secure**: Robust authentication, role-based access control, and API key management.
- **Flexible**: Support for boolean, string, number, and JSON flags with targeted rules.
- **Developer Focused**: First-class SDKs and a comprehensive API.
- **Workflows**: Built-in change request process for critical environments.
